## рҹҺҜ What is this?

An **AI-powered shopping assistant** built for [Maa Kaali Creations](https://maakaalicreations.in) вҖ” a real Indian ethnic wear e-commerce store with **1000+ happy customers**.

Instead of browsing through hundreds of products, customers simply **chat** with the AI вҖ” in their own language вҖ” and get personalized saree, lehenga, and suit set recommendations instantly.

> **"Show me red silk saree under вӮ№1500"** вҶ’ AI recommends the best matching products from 633+ catalog

Powered by **osmAPI's Qwen 3.5 model** for intelligent, context-aware conversations.

## рҹҡҖ Quick Start

```bash
# Clone the repo
git clone https://github.com/Shubh2-0/maa-kaali-ai-assistant.git
cd maa-kaali-ai-assistant

# Install dependencies
npm install

# Add your osmAPI key
echo "OSM_API_KEY=your_key_here" > .env

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)
